# 沉没成本陷阱 - 框架图

Professional hand-drawn sketchnote framework on pure white background. Black ink line work with slight wobble, à la Mike Rohde sketchnoting.

## STICKY NOTE STYLE
Main content on a large pink sticky note (#F8BBD9, slight shadow, -3-degree tilt). Warning theme throughout.

## LAYOUT: Trap Visualization + Escape Actions

## TOP SECTION: THE TRAP (Pink sticky note)
**标题**: 大厂老员工的"沉没成本陷阱"

**四大枷锁** (Four chains around a stick figure):
1. 🔒 股票还没 vest 完（等行权窗口）
2. 🔒 房贷绑定现金流（无法接受降薪）
3. 🔒 生活方式锚定大厂薪资（降薪=断崖）
4. 🔒 履历过于"大厂化"（外面看不懂）

Visual: Stick figure trapped in a box with four chains

**心理状态**: "再等等"、"再撑撑" → 直到选择窗口关闭

## BOTTOM SECTION: THE ESCAPE (Green sticky note #C8E6C9)
**标题**: 对冲策略（现在就开始，不是辞职）

**四个行动**:
1. ✓ 维护外部人脉网络（不只是猎头）
2. ✓ 保持市场竞争力感知（知道自己的"市价"）
3. ✓ 降低财务杠杆（提前还贷、降低固定支出）
4. ✓ 建立组织外"作品集"（开源、博客、影响力）

Visual: Stick figure breaking chains, stepping toward open door

## CENTER ARROW
Large hand-drawn arrow from trap to escape with label: "理性的做法是忽略沉没成本，看未来预期"

## COLORS (mono-ink palette with warning theme)
- Pure White background (#FFFFFF)
- Near Black (#1A1A1A) for all lines, text
- Pink sticky note (#F8BBD9) for trap section
- Green sticky note (#C8E6C9) for escape section
- Coral Red (#E8655A) for 🔒 chains and warning
- Muted Teal (#5FA8A8) for ✓ escape actions

## ELEMENTS
- Two contrasting sticky notes (trap vs escape)
- Chain icons for constraints
- Checkmark icons for solutions
- Stick figure transformation (trapped → free)
- Large directional arrow in center

## STYLE
Hand-lettered headings, handwritten body text, generous white space. Visual contrast between the "trapped" pink section and "free" green section.
