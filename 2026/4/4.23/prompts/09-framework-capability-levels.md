# 能力层次模型 L1-L5 - 框架图

Professional hand-drawn sketchnote framework on pure white background. Black ink line work with slight wobble, à la Mike Rohde sketchnoting.

## STICKY NOTE STYLE
Main content on a large yellow sticky note (#FFF9C4, slight shadow, 5-degree tilt). Five stacked horizontal layers from bottom to top.

## LAYOUT: Hierarchical Pyramid Structure

## LEVELS (from bottom to top)

### L1: 能完成任务 (Bottom layer - widest)
- Label: "拿到明确指令后能做完"
- Icon: Checkmark in box
- Sticky note shade: Lightest yellow
- Status: "AI 正在替代"

### L2: 能执行方案
- Label: "拿到方案后能高质量实现"
- Icon: Gear/cog
- Sticky note shade: Light yellow
- Status: "AI 正在替代"

### L3: 能设计方案
- Label: "拿到问题后能给出最优解"
- Icon: Lightbulb
- Sticky note shade: Medium yellow
- Status: "P8-P9 需要"

### L4: 能定义问题
- Label: "从混沌中找到最值得做的事情"
- Icon: Magnifying glass
- Sticky note shade: Darker yellow
- Status: "P8-P9 需要"

### L5: 能定义赛道 (Top layer - narrowest)
- Label: "这个领域应该怎么走，终局是什么"
- Icon: Crown/star
- Sticky note shade: Darkest yellow
- Status: "稀缺"

## ANNOTATIONS
- Arrow pointing to L1-L2: "AI 正在让这两层变得廉价"
- Arrow pointing to L3-L5: "P8-P9 需要的是这里"

## COLORS (mono-ink palette with yellow theme)
- Pure White background (#FFFFFF)
- Near Black (#1A1A1A) for all lines, text
- Yellow sticky note (#FFF9C4) for main frame
- Coral Red (#E8655A) for "AI 正在替代" label
- Muted Teal (#5FA8A8) for "P8-P9 需要" label

## ELEMENTS
- Main yellow sticky note as container
- Five stacked sticky note layers
- Hand-drawn arrows between layers pointing upward
- Small doodle icons per level
- Hand-lettered title "能力层次模型" at top

## STYLE
Hand-lettered headings (bold, oversized), handwritten body annotations, generous white space around the sticky note, no computer fonts, no gradients. The sticky note should look like it was hand-placed on a whiteboard.
