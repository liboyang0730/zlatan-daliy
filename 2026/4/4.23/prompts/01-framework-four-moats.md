# 四层护城河模型 - 框架图

Professional hand-drawn sketchnote framework on pure white background. Black ink line work with slight wobble, à la Mike Rohde sketchnoting.

## STICKY NOTE STYLE
Main content displayed on a large yellow sticky note (#FFF9C4, slight shadow, 5-degree tilt). The sticky note has a hand-torn edge effect and appears to be pinned to a white board.

## LAYOUT: Hierarchical Pyramid Structure
Four stacked horizontal layers from bottom to top, each on its own smaller sticky note (yellow variant shades).

## ZONES
- **Bottom Layer (Base)**: 效率护城河（生存线）
  - Label: "6个月内必须建立"
  - Key points: 成为团队AI原生标杆、建立个人AI工作流SOP、量化效率提升
  - Warning icon: ⚠️ 效率悖论

- **Second Layer**: 认知护城河（竞争线）
  - Label: "6-12个月建立"
  - Key points: 成为领域专家、培养问题定义能力、建立商业感知力

- **Third Layer**: 关系护城河（发展线）
  - Label: "持续建设"
  - Key points: 向上管理、横向影响、打造个人品牌

- **Top Layer (Apex)**: 不对称护城河（终极线）
  - Label: "长期建设"
  - Key points: 成为"AI+领域"桥梁人、积累隐性知识、创造独特协调项目

## COLORS (mono-ink palette with sticky note yellow)
- Pure White background (#FFFFFF)
- Near Black (#1A1A1A) for all lines, text
- Sticky Note Yellow (#FFF9C4) for main frame
- Coral Red (#E8655A) for "⚠️ 效率悖论" warning only
- Muted Teal (#5FA8A8) for "长期建设" emphasis only

## ELEMENTS
- Main yellow sticky note as container
- Four smaller sticky notes stacked vertically
- Hand-drawn arrows between layers pointing upward
- Small doodle icons per layer (lightning bolt for efficiency, brain for cognition, people for relationships, crown for asymmetry)
- Hand-lettered title "个人护城河四层模型" at top

## STYLE
Hand-lettered headings (bold, oversized), handwritten body annotations, generous white space around the sticky note, no computer fonts, no gradients. The sticky note should look like it was hand-placed on a whiteboard.
