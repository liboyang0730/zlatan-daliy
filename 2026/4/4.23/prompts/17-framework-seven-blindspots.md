# 七个需要思考的问题 - 框架图

Professional hand-drawn sketchnote framework on pure white background. Black ink line work with slight wobble, à la Mike Rohde sketchnoting.

## STICKY NOTE STYLE
Seven small sticky notes arranged in a grid pattern, each with a different color theme.

## LAYOUT: Seven Points Grid (2 rows, 4 columns with one empty)

### 问题 1 (Yellow sticky note #FFF9C4)
**标题**: 转型成功的条件
**内容**: 并非所有人都能成功转型
**对策**: 找到你的专精点

### 问题 2 (Orange sticky note #FFE0B2)
**标题**: 组织因素
**内容**: 位置有时比能力重要
**对策**: 做被看见的事

### 问题 3 (Pink sticky note #F8BBD9)
**标题**: AI 形式主义
**内容**: Demo ≠ 真正的价值
**对策**: 做实践者

### 问题 4 (Blue sticky note #BBDEFB)
**标题**: 团队协作
**内容**: 个人优化可能影响团队
**对策**: 成为力量倍增器

### 问题 5 (Green sticky note #C8E6C9)
**标题**: 技术不确定性
**内容**: AI 发展速度难预测
**对策**: 保持反脆弱性

### 问题 6 (Purple sticky note #E1BEE7)
**标题**: 收入变化
**内容**: 市场供需在变化
**对策**: 建立多元收入

### 问题 7 (Gray sticky note #E0E0E0)
**标题**: 框架局限
**内容**: 无法预测所有情况
**对策**: 从小事开始行动

## CENTER CALLOUT
"任何框架都有局限性"

## COLORS (mono-ink palette with multi-color theme)
- Pure White background (#FFFFFF)
- Near Black (#1A1A1A) for all lines, text
- Yellow (#FFF9C4) for Point 1
- Orange (#FFE0B2) for Point 2
- Pink (#F8BBD9) for Point 3
- Blue (#BBDEFB) for Point 4
- Green (#C8E6C9) for Point 5
- Purple (#E1BEE7) for Point 6
- Gray (#E0E0E0) for Point 7

## ELEMENTS
- Seven sticky notes in grid layout
- Number icons (1-7) per sticky note
- Brief content per point
- Counter-measure per point

## STYLE
Hand-lettered titles, handwritten summaries, generous white space.
