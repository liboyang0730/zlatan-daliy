# 三个底层认知 - 框架图

Professional hand-drawn sketchnote framework on pure white background. Black ink line work with slight wobble, à la Mike Rohde sketchnoting.

## STICKY NOTE STYLE
Three large sticky notes arranged horizontally, each containing one core cognition. Blue theme with slight variations.

## LAYOUT: Three-Column Framework

## COGNITION 1 (Blue sticky note #BBDEFB)
**Title**: 认知一
**Core Message**: 你不是在跟 AI 竞争，你是在跟"AI+其他人"竞争

**Visual**:
- Stick figure on left (你)
- VS symbol in middle
- Stick figure with robot icon on right (会用AI的人)
- Arrow pointing: 真正的威胁

**Insight**: 问题不是"AI 能不能做我的工作"，而是"用了 AI 的人能不能做我的工作"

## COGNITION 2 (Lighter blue sticky note #90CAF9)
**Title**: 认知二
**Core Message**: 稀缺性来自组合，不是单一技能

**Formula Box**:
```
技术深度 × 业务理解 × 沟通能力 × AI驾驭力 = 不可替代性
```

**Visual**:
- Four icons in a row: wrench (技术), briefcase (业务), speech bubble (沟通), robot (AI)
- Multiplication signs between them
- Equals sign pointing to star (不可替代性)

**Insight**: 每个单项都有很多人具备，但四项的乘积是独一无二的

## COGNITION 3 (Darker blue sticky note #64B5F6)
**Title**: 认知三
**Core Message**: 最好的防守是进攻

**Comparison**:
- Left (防守): 守城的人 → 维护老系统、做确定性的工作
- Right (进攻): 开疆的人 → 探索新方向、在不确定中创造价值

**Arrow**: "不要做守城的人，要做开疆的人"

## COLORS (mono-ink palette with blue theme)
- Pure White background (#FFFFFF)
- Near Black (#1A1A1A) for all lines, text
- Blue sticky notes in gradient (#BBDEFB, #90CAF9, #64B5F6)
- Coral Red (#E8655A) for warning/防守
- Muted Teal (#5FA8A8) for positive/进攻

## ELEMENTS
- Three large sticky notes side by side
- Hand-drawn icons per cognition
- Stick figures showing different scenarios
- Formula box for cognition 2
- VS comparison for cognition 3

## STYLE
Hand-lettered cognition titles (认知一、认知二、认知三), handwritten insights, generous white space. Each sticky note is a self-contained concept card.
