# Plan B 五条路径 - 流程图

Professional hand-drawn sketchnote flowchart on pure white background. Black ink line work with slight wobble, à la Mike Rohde sketchnoting.

## STICKY NOTE STYLE
Main content on a large yellow sticky note (#FFF9C4). Five paths branch out from a central decision point, each path on its own colored sticky note.

## LAYOUT: Branching Flow from Center

## CENTER NODE
Central sticky note (white with border): "Plan B 路径选择"
Question: "离开大厂后，我能做什么？"

## BRANCHING PATHS

### Path 1: 中小公司"全栈高管" (Blue sticky note #BBDEFB)
- Arrow from center
- Description: 大厂 P7 在外面可能是 CTO 级别
- 适合人群: 35岁左右、有管理经验、不怕脏活累活的 P7-P8
- Reality check: 薪资降 30-50%，但自由度高

### Path 2: 技术咨询与顾问 (Green sticky note #C8E6C9)
- Arrow from center
- Description: "懂技术又懂业务的人来告诉他们怎么落地"
- 适合人群: 有垂直领域深度积累、沟通能力强的 P7+
- Reality check: 起步收入不稳定，需 6-12 月建立客户

### Path 3: 独立开发者/小团队创业 (Pink sticky note #F8BBD9)
- Arrow from center
- Description: AI 降低产品开发门槛，1-2人+AI 可做产品
- 适合人群: 有行业洞察、愿意做 sales 工作
- Reality check: 技术不是瓶颈，获客才是

### Path 4: 技术教育与内容 (Yellow sticky note #FFF9C4)
- Arrow from center
- Description: "教别人在具体工作场景中用 AI 重构工作流"
- 适合人群: 有表达欲、能把复杂事情讲清楚

### Path 5: AI 不易替代方向 (Orange sticky note #FFE0B2)
- Arrow from center
- Directions: 安全与合规、基础设施/SRE、AI 基建、硬件相关
- Why: 涉及法律/政策/风险判断，需要人兜底
